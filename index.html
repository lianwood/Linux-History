How the Linux works

Linux is an operating system. It manages all the hardware resources associated with your desktop/laptop.
The Linux has 7 parts to it.

  • The first part is called the Bootloader. This manages the boot process of your computer.
  • Second is the Kernel. This part is actually the Linux. It is the core of the system and manages the CPU, memory, and peripheral devices.
  • Third is the Daemons. They are background services (printing, sound, scheduling) that start after you log in or start during boot.
  • Fourth is the Shell. The Shell is a command process that allows you to control your computer by using commands.
  • Fifth is the Graphical Server. It is a sub-system that displays the graphics on your monitor.
  • Sixth is the Desktop Environment. There are many to choose from (Unity, GNOME, Cinnamon, Enlightenment, KDE, XFCE etc.) Each one includes built-in applications ( web browsers, games, configuration tools, and file managers)
  • The Seventh part is the applications. Desktop Environments don't offer a full aray of apps. Linux offers software titles that can be easily found and installed. For example, the App Store.
